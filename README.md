# 基于时间轴切换的世界地图

#### 介绍
可通过时间轴切换的世界政权疆域图，包含公元1200-2000年间八个时段，内附各行政区域历史简介信息，基于html,css和js开发。

#### 安装教程

下载源码即可

```
git clone https://gitee.com/ywc2023/webfinalwork.git
```


#### 使用说明

1. 点击下方时间轴实现地图切换
2. 点击具体区域弹出文本介绍框
3. 鼠标左键按下可平移地图
4. 鼠标滚轮可缩放地图
5. 鼠标悬停区域变色
